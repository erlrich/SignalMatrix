# SignalMatrix QGIS Plugin

[![License](https://img.shields.io/badge/license-Commercial-blue)](LICENSE)

SignalMatrix is a professional QGIS plugin for RF engineers and network optimization professionals. It processes measurement data to enrich building footprints with RSRP values.

This plugin integrates coverage measurement data with OSM building layers for building-level RF analysis.

- Multi-operator support (customizable)
- Administrative boundary filtering
- Statistical analysis and benchmarking
- Export to CSV/XLSX
